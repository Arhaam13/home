What is slight of hand?

Slight of hand is a general term used to describe all the maneuvers done with a deck of cards. It is an essential skill one must master to do any type of card trick because every trick has some type of slight of hand. The most common ones are as follows:

  • The Break
  • False cut
  • False shuffle
  • The Undercut
  • Double Lift
